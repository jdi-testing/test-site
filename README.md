# vuetify-site

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Launch server for production
```
npm run start
```

### Lints and fixes files
```
npm run lint
```
